# TikTok → Reels Downloader (Max Quality)

Node.js utility that automates yt-dlp to download TikTok videos
in the highest available quality (up to 1080p), optimized for
Instagram Reels.

## Requirements
- Node.js
- yt-dlp
- ffmpeg
- Google Chrome (logged into TikTok)

## Usage
```bash
node baixarTikTokA.js <TIKTOK_VIDEO_URL>
